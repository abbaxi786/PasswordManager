# PasswordManager

PasswordManager is a secure web application used for saving passwords of different accounts and websites.  
It uses encryption and JWT tokens for security, and Next.js for full-stack development.  
MongoDB is used for database storage.

---

## Tech Stack

- **Next.js** — full-stack framework (server + client components)
- **MongoDB / Mongoose** — data storage
- **JWT Authentication** — secure login & access control
- **Encryption** — protects stored passwords
- **DaisyUI + TailwindCSS** — UI styling
